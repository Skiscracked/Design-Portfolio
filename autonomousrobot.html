Autonomous Robot

Overview

As part of a team project for the ECED 3901 course at Dalhousie University, I led the development of an autonomous robot designed to navigate a spice field, avoid obstacles, and collect items under strict competition conditions. The robot successfully demonstrated key principles of robotic design, mapping, and autonomous navigation.

Project Goals

  • Autonomous Navigation: Enable the robot to efficiently navigate a dynamic environment using waypoints and avoid obstacles.
  • Item Collection: Implement a reliable system for detecting and collecting "spices" using optical signaling patterns.
  • Competition Readiness: Ensure compliance with size, voltage, and safety constraints while meeting competition requirements.

My Role and Contributions

  • Project Leadership:
    • Led the integration of various subsystems and ensured seamless functionality.
    • Organized team efforts, assigned tasks, and tracked progress using a Gantt chart.
  • Launch File Development:
    • Designed and implemented ROS2 launch files to initialize all robot nodes.
    • Configured the Nav2 library, AMCL (Adaptive Monte Carlo Localization), waypoint follower, and competition start nodes.
    • Ensured the system subscribed to relevant topics for communication.
  • Waypoint Navigation and Mapping:
    • Created YAML files to define waypoints, including parameters like coordinates, headings, and timeout settings.
    • Integrated LiDAR and IMU sensors with the navigation system for accurate localization and obstacle avoidance.
    • Employed the SMAC2D planner to optimize paths dynamically, adapting to obstacles in real time.
  • Obstacle Avoidance:
    • Configured algorithms to detect and respond to "worm" obstacles using LiDAR data.
    • Fine-tuned avoidance strategies to minimize delays and maximize competition performance.

Technical Highlights

  • Tools & Technologies: ROS2, LiDAR, IMU, Arduino Nano, Python, C++, YAML, and Beelink Mini PC.
  • Sensors & Actuators: Integrated LiDAR for mapping, IMU for orientation, and bright LED strips for item collection detection.
  • Budget Efficiency: Completed the project using only $80 of the $300 budget, demonstrating effective resource management.

Challenges & Solutions

  • Startup Issues: Encountered inconsistent startup behaviors during competition rounds. Resolved through improved launch file sequencing and systematic testing.
  • Dynamic Obstacle Navigation: Adapted navigation algorithms to handle unexpected movements of "worm" obstacles.
  • Integration Testing: Conducted rigorous verification tests for all subsystems to ensure reliability and seamless performance.

Results

  • Achieved the second-highest score for item collection in the competition.
  • Demonstrated the ability to autonomously navigate the field, avoid obstacles, and complete tasks within the allotted time.
  • Garnered valuable lessons for improving startup reliability and dynamic responsiveness in future iterations.
